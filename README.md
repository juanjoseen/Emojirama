# Emojirama
Un simple pero divertido juego de memoria basado en emojis

El juego consta de 3 niveles:
* Fácil (4x4)
* Medio (4x6)
* Difícil (5x6)

El reto está en encontrar todos los pares antes de que termine el tiempo
